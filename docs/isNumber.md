# isNumber

提供一个方便的方式来判断一个值是否为数字类型

## 引入方式

<!-- ### CJS

```javascript
const { isNumber } = require("tool-assistant");
``` -->

### ESM

```javascript
import { isNumber } from "tool-assistant;
```

## 示例

```javascript
// 引入 isNumber 函数
import { isNumber } from "tool-assistant";

// 定义一个变量
const value = 123;

// 使用 isNumber 函数判断值是否为数字类型
const result = isNumber(value);

// 打印结果
console.log(result); // 输出 true
```
