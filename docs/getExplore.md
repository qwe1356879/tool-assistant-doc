# getExplore

函数用于判断用户的浏览器并返回浏览器的名称和版本。

## 引入方式

<!-- ### CJS

```javascript
const { getExplore } = require("tool-assistant");
``` -->

### ESM

```javascript
import { getExplore } from "tool-assistant;
```

## 示例

```javascript
import { getExplore } from "tool-assistant";

var explore = getExplore();

console.log(explore);
```
