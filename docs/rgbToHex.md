# rgbToHex

将 RGB 颜色转换成 16 进制颜色

## 引入方式

<!-- ### CJS

```javascript
const { rgbToHex } = require("tool-assistant");
``` -->

### ESM

```javascript
import { rgbToHex } from "tool-assistant;
```

## 示例

```javascript
import { rgbToHex } from "tool-assistant";

function _yourFun() {
  let hex = rgbToHex("rgb(255,255,255)");
}
```
