# buildUUID

在 JavaScript 中生成随机的 UUID

## 引入方式

### CJS

```javascript
const { buildUUID } = require("tool-assistant");
```

### ESM

```javascript
import { buildUUID } from "tool-assistant;
```

## 示例

```javascript
// 引入 buildUUID 函数
import { buildUUID } from "tool-assistant";

// 生成随机的 UUID
const uuid = buildUUID();

// 打印生成的 UUID
console.log(uuid);
```
