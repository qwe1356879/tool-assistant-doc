# hexToRGB

将 16 进制颜色转换成 RGB 颜色

## 引入方式

### CJS

```javascript
const { hexToRGB } = require("tool-assistant");
```

### ESM

```javascript
import { hexToRGB } from "tool-assistant;
```

## 示例

```javascript
import { hexToRGB } from "tool-assistant";

function _yourFun() {
  let rgb = hexToRGB("#ffffff");
}
```
